# Creative-Showcase
A responsive web app where artists register, log in, upload artwork, manage private profiles, and showcase public galleries in dynamic mosaic layouts.

Cricket World Cup Insights: This Google Looker Studio dashboard analyzes Cricket World Cup data to compare team performances. It uses interactive filters to enable side-by-side comparison of teams based on runs scored and wickets taken. The dashboard helps identify performance patterns, strengths, and differences across teams through clear visualizations.
